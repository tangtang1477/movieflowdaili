
# 还原 MovieFlow 代理商门户前端样式

## 设计分析

页面采用温暖的大地色调设计：
- **背景**：浅米色/沙色渐变
- **卡片**：左右分栏布局，圆角卡片，带阴影
- **左侧**：深棕色到黑色渐变背景，白色大标题文字，带标签徽章（如 "AGENT SIGNUP"）
- **右侧**：白色/奶白色背景，表单区域
- **按钮**：赤陶/铜色渐变按钮，圆角
- **链接**：赤陶色文字链接
- **输入框**：浅灰色边框，圆角

## 实现内容

### 1. 更新设计主题 (`src/styles.css`)
- 设置暖色调配色方案：沙色背景、赤陶色主色、深棕色等

### 2. 创建共享布局组件
- `src/components/AuthLayout.tsx` — 左右分栏卡片布局（左侧深色背景+标语，右侧白色表单区）

### 3. 创建登录页 (`src/routes/login.tsx`)
- 标题 "代理商登录"
- 邮箱、密码输入框
- "登录并进入控制台" 按钮
- "没有账号？去注册" 链接

### 4. 创建注册页 (`src/routes/register.tsx`)
- 标题 "代理商注册"
- 双列表单：负责人姓名、登录邮箱、代理商企业名称、邀请码前缀、联系人、联系电话、联系邮箱、密码、确认密码
- "创建代理商账号" 按钮
- "已有账号？去登录" 链接

### 5. 更新首页 (`src/routes/index.tsx`)
- 跳转到登录页或作为入口

## 技术细节

- 纯前端样式还原，不涉及后端逻辑
- 使用 Tailwind CSS 语义化 token
- 表单仅做样式展示，不接入 API
- 路由：`/login`、`/register`、`/`（重定向到登录）
